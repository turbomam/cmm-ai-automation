## Error Type Descriptions

- **contains_space**: CURIE contains whitespace (invalid)
- **missing_colon**: No colon separator found
- **invalid_format**: Does not match CURIE pattern prefix:local_id
- **empty_prefix**: Prefix before colon is empty
- **empty_local_id**: Local ID after colon is empty
- **multiple_colons**: Contains more than one colon (may be valid but flagged)
- **url_not_curie**: Full URL instead of compact CURIE (technically valid but not compact)
