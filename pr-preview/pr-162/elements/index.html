    CMM AI Automation Schema

    Schema for Critical Mineral Metabolism (CMM) data curation.

    Models microbial strains, growth media, ingredients, and their relationships for knowledge graph generation.

    URI: https://w3id.org/turbomam/cmm-ai-automation

    Name: cmm-ai-automation

    Classes

    Class Description
    CMMDatabase Container for all CMM entities
    CrossReference A cross-reference to an external database or identifier
    DataConflict Records conflicts between data sources for reconciliation review
    EnrichedIngredient A chemical entity enriched with data from multiple sources
    Genome A genome assembly from NCBI GenBank/RefSeq or other sources
    GrowthPreference Reified relationship: a strain's growth in a specific medium
    IngredientComponent Reified relationship: an ingredient as used in a mixture, with concentration ...
    NamedThing A generic grouping for any identifiable entity
            Ingredient A chemical entity that can be a component of solutions or media
            Mixture Abstract base class for things composed of ingredients
                    GrowthMedium A complete formulation for cultivating microorganisms
                    Solution A pre-made concentrated mixture of ingredients, typically diluted into media
    SolutionComponent Reified relationship: a solution as used in a medium, with volume/dilution
    SourceRecord Tracks the provenance of data from a specific source
    Strain A microbial strain enriched with data from multiple sources
    Taxon A taxonomic entity from NCBI Taxonomy
